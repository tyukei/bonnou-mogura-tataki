# 煩悩めくり

めくれた札が **煩悩** なら「読経」で撃ち払い、**仏教語** は見送る——
判断力を試す和風カードめくりアクション。

▶ **プレイ: https://tyukei.github.io/bonnou-mogura-tataki/**

## 遊び方

- 3×3 の札が次々と自動でめくれる
- **煩悩（赤枠）** … 表になっている間に「読経」で撃つと浄化。連鎖でスコアアップ
- **仏教語（緑枠）** … 撃つと罰。見送れば徳(MP)が貯まる
- 煩悩を見逃すと襲われて精神力が減少。精神力が尽きると敗北
- 照準は自動。目標数の煩悩をすべて打ち払えばクリア
- 操作: **読経ボタン** / PC は **Space キー**

## 構成

プレーンな HTML / CSS / バニラ JS（ビルド不要）。

| ファイル | 役割 |
|---|---|
| `index.html` | 画面構造 |
| `style.css` | スタイル・アニメーション |
| `game.js` | ゲームロジック・効果音 (WebAudio) |
| `assets/monk_back_1.png` | 僧侶スプライト |

ローカルで動かす場合:

```bash
python3 -m http.server 8000
# http://localhost:8000 を開く
```
